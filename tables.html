HTML Tables

The<table> tag defines an HTML table.

Each table row is defined with a <tr> tag. Each table header is defined with a <th> tag. Each table data/cell is defined with a <td> tag.

By default, the text in <th> elements are bold and centered.

By default, the text in <td> elements are regular and left-aligned.

Example

A simple HTML table:

<table >
  <tr>
  <th>Sr No</th>
   <th>Subject<th>
   <th>Marks</th>
</tr> 
<tr>   
  <td>1</td>
   <td>English</td>
    <td>84</td>
  </tr>
  <tr>
    <td>2</td>
    <td>Maths</td>
    <td>92</td>
  </tr>
</table>

Output:

Sr No Subject Marks
1 English 84
2 Maths 92

HTML Table - Add a Border


You can use border attribute of table tag in HTML to specify border.

Example

<table border=1 >
  <tr>
  <th>Sr No</th>
   <th>Subject<th>
   <th>Marks</th>
</tr> 
<tr>   
  <td>1</td>
   <td>English</td>
    <td>84</td>
  </tr>
  <tr>
    <td>2</td>
    <td>Maths</td>
    <td>92</td>
  </tr>
</table>

Output:

Sr No Subject Marks
1 English 84
2 Maths 92

HTML Table - Cell that Spans Many Columns


To make a cell span more than one column, use the colspan attribute:

Example

<table>
  <tr>
    <th>Name</th>
    <th colspan="2">Telephone</th>
  </tr>
 <tr>
    <td>Lovish</td>
    <td>70151*****</td>
    <td>98765*****</td>
  </tr>
</table>

Output:

Name Telephone
Lovish 70151***** 98765*****

HTML Table - Cell that Spans Many Rows


To make a cell span more than one row, use the rowspan attribute:

Example

<table >
 <tr>
    <th>Name:</th>
    <td>Bill Gates</td>
 </tr>
  <tr>
    <th rowspan="2">Telephone:</th>
   <td>98765*****</td>
 </tr>
 <tr>
   <td>70151*****</td>
  </tr>
</table>

Output:

Name: Lovish
Telephone: 98740*****
70151*****
